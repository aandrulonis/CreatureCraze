# A WIP collection of mini-games

Completed Games:
* A vanilla JS rocket video game, with simplified physics modelling
* A vanilla JS version Pong with customized graphics
* A vanilla JS Space Invaders with customized graphics

In Progress:
* Custom C++ graphics calculations
* 6-DOF C++ motorcycle simulation
* Custom C++ genetic algorithm for song generation
* A vanilla JS Pacman with randomized maze generation
* User database

C++ is integrated using the Enscriptem toolchain for WASM.
